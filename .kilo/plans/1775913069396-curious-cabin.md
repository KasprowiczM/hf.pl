Add to SEO.jsx:
```javascript
{

  "@type": "BreadcrumbList",
  "itemListElement": [{
    "@type": "ListItem",
    "position": 1,
    "name": "hf.pl",
    "item": "https://hf.pl/"
  }]
}
```

### 1.5 Add missing OG tags

```javascript
  "url": "https://hf.pl",
  "sameAs": [
    // TODO: Add social profile URLs here

  ]

```
**Action needed**: Provide social profile URLs to include in `sameAs` array

### 2.2 WebSite Schema with SearchAction
Enable Sitelinks Search Box in Google:
```javascript
{
  "@type": "WebSite",
  "name": "hf.pl",
  "url": "https://hf.pl",
  "potentialAction": {
    "@type": "SearchAction",
    "target": "https://hf.pl?q={search_term_string}",
    "query-input": "required name=search_term_string"
  }
}
```

### 2.3 Content Structure for AI Extraction
- Add "Answer-first" paragraphs after each H2 heading (direct answers)
- Include statistics with named sources (e.g., "According to Verisign 2025...")
- Add datePublished/dateModified to page metadata

### 2.4 Review Schema for Trust Signals
Add AggregateRating schema (even if simulated):
```javascript
{
  "@type": "AggregateRating",
  "ratingValue": "4.9",
  "reviewCount": "50",
  "bestRating": "5"
}
```
